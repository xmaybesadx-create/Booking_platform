# Polyglot Booking Platform

Проєкт демонструє підхід **Polyglot Microservices**: два незалежні бекенди (Java + Python) та один спільний фронтенд (React), об'єднані у масштабовану систему бронювання апартаментів та оренди офісів.

---

## 1️⃣ Архітектура системи

**Загальна схема взаємодії:**

1. Користувач відкриває сайт (React).
2. React звертається до **Python-сервісу**, щоб знайти апартаменти/офіси.
3. Коли користувач натискає «Забронювати», React звертається до **Java-сервісу**.
4. Java-сервіс перевіряє дати, проводить бронювання через **Workflow** і зберігає результат.

**Компоненти:**

| Компонент | Стек | Роль |
|-----------|------|------|
| Frontend | React + Tailwind CSS | UI: сторінки пошуку, апартаментів, особистий кабінет |
| Catalog Service | Python 3.10+, FastAPI, SQLAlchemy, PostgreSQL | Пошук та управління апартаментами, медіафайли, фільтри |
| Booking Service | Java 17/21, Spring Boot, Spring Data JPA, PostgreSQL | Логіка бронювання, Workflow, перевірка перетину дат, concurrency handling |

---

## 2️⃣ Frontend (React + Tailwind)

**Сторінки:**

- **Пошук:** отримує дані з Python-сервісу (список апартаментів, фільтри).
- **Апартаменти:** показує фото та календар доступності (дані з Python + Java).
- **Особистий кабінет:** статус бронювань (Java).

**Запуск локально:**

```bash
cd frontend
npm install
